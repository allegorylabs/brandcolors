## Introduction

Official Brand Colors Collection.

## Installation

```shell
npm install @brandcolors/core
```

## Usage

### Styles

```scss
@use "@brandcolors/core" as brandcolors;

.foo {
    color: brandcolors.$facebook;
}
```
