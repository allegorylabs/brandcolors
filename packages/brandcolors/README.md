## Introduction

Brand colors collection.

## Installation

```shell
npm install brandcolors
```

## Usage

```scss
@use "brandcolors";

.foo {
    color: brandcolors.$twitter-primary;
    // color: #1da1f2;
    color: brandcolors.$twitter-primary-rgb;
    // color: rgb(29, 161, 242);
}
```
